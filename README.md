# movie-review-sentiment-analysis
A machine learning based sentiment analysis project that classifies movie reviews as positive or negative using Natural Language Processing (NLP) techniques.

Project Overview
*Movie reviews contain valuable insights about audience sentiment.
*This project uses NLP and Machine Learning to:
*Process text data
*Extract meaningful features
*Train a sentiment classification model
*Predict sentiment from unseen reviews

Objectives
✔ Perform sentiment analysis on text data
✔ Apply NLP preprocessing techniques
✔ Train ML model for classification
✔ Predict positive or negative sentiment
✔ Visualize text data patterns

Features
*Text preprocessing (cleaning & tokenization)
*Stopword removal
*Feature extraction (TF-IDF / Bag of Words)
*Sentiment classification
*Model evaluation
*Prediction on custom reviews

Technologies Used
*Python
*Natural Language Processing (NLP)
*Scikit-learn
*Pandas
*NumPy
*Matplotlib / Seaborn
